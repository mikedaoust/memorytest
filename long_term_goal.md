# Long-Term Goal
I want to build a self-hosted assistant that runs locally on my MacBook Pro M4 Max with 128GB of RAM.

The long-term objective is for this assistant to support human-like episodic memory, so it can retain, organize, and recall past interactions in a useful and coherent way over time.

I also want a web interface and/or mobile app so I can communicate with the assistant from both my MacBook and iPhone.
